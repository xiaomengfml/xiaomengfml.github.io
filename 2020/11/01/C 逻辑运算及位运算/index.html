Ⅲ逻辑运算符

逻辑运算符适合多个判断一起。

配合关系运算符一起进行判断。

&&、||、!

假设int A = 1，int B = 0，

运算符 描述 实例
&& 称为逻辑与运算符。如果两个操作数都非零，则条件为真 (A&&B)为假
|| 称为逻辑或运算符。如果两个操作数中有任意一个非零，则条件为真 (A||B)为真
! 称为逻辑非运算符。用来逆转操作数的逻辑状态。如果条件为真则逻辑非运算符将使其为假 !(A && B) 为真

Ⅵ移位运算符

按位进行左移右移，可理解成简单乘法和除法。

假设int A = 60，int B = 13，则：

运算符 描述 实例
<< 二进制左移运算符。将一个运算对象的各二进制位全部左移若干位（左边的二 进制位丢弃，右边补0） A << 2 将得到240，即 为 1111 0000
>> 二进制右移运算符。将一个数的各二进 制位全部右移若干位，正数左补0，负数左补1，右边丢弃 A >> 2 将得到15，即 为 0000 1111

位运算符是非常快的，因为计算机底层都是二进制，位运算符直接对二进制进行操 作，效率非常高。
